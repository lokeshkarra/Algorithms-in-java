# Sorting
This repo contains the Implementation of all sorting algorithms in Java.
1) Bubble Sort
2) Selection sort
3) Insertion sort
4) Merge sort
5) Quick sort

To implement these files, run the following commands in cmd.

 -> Eg   : To implement BubbleSort.java
 -> cmd  :  javac BubbleSort.java
 -> cmd  :  java BubbleSort
